# Screen Pet Extension - Privacy Policy & Permissions

**Last Updated:** December 2025  
**Extension Name:** Screen Pet Extension  
**Version:** 1.0

---

## Privacy Policy

### Data Collection
This extension does **NOT** collect, transmit, or share any personal data. All information is stored locally on your device.

### Local Storage
The extension uses Chrome's local storage API to save:
- Your coin count
- Purchased items (hats and accessories)
- Currently equipped items
- Your selected pet preference

This data:
- ✅ Stored locally on your device
- ✅ Never transmitted to external servers
- ✅ Never shared with third parties
- ✅ Can be cleared by uninstalling the extension

### Network Activity
This extension makes **ZERO** network requests. It:
- Does not connect to any external servers
- Does not send analytics or telemetry
- Does not track your browsing
- Operates 100% offline after installation

## Permissions Explanation

### Required Permissions

#### 1. "storage" Permission
**Why we need it:** To save your game progress locally
**What it does:**
- Saves your collected coins
- Remembers which accessories you've purchased
- Stores which items are currently equipped
- Remembers your selected pet

**What it does NOT do:**
- Does not access any other Chrome data
- Does not sync to external servers
- Does not track your activity

#### 2. "<all_urls>" Match Pattern
**Why we need it:** Core functionality - pets appear on all websites
**What it does:**
- Displays your chosen pet on webpages you visit
- Allows the pet to move around and interact on any site
- Enables coin collection across all pages

**What it does NOT do:**
- Does not read page content or user data
- Does not modify page content
- Does not track which sites you visit
- Only adds a visual overlay (your pet)

**Why it's broad:**
Desktop pets are meant to follow you everywhere - just like classic desktop companions (Neko, eSheep). Limiting to specific sites would break the core user experience.

## Single Purpose Statement

**The extension's sole purpose is:** To provide animated desktop pet companions that roam across the user's browser as a fun, nostalgic experience.

**This includes:**
- Displaying 6 different animated pets
- Allowing users to interact with pets using keyboard/mouse
- Providing a coin collection mini-game
- Offering cosmetic customization via an item store

**This does NOT include:**
- Data collection or analytics
- Content modification
- Ad injection
- Tracking or monitoring
- Any functionality beyond desktop pet entertainment

## User Control

Users have full control:
- **Enable/Disable:** Via Chrome extension toggle
- **Hide Pet:** Button in popup
- **Remove Coins:** Toggle button in popup
- **Reset Data:** Uninstall extension to clear all data
- **Change Pets:** Select different pet anytime
